# 📌 Introduction to CSS

**CSS (Cascading Style Sheets)** is used to **style and design web pages**.  
It controls how HTML elements look on the screen.

---

## 🔑 What CSS Can Do

- Change **colors** (text, background)  
- Set **fonts and sizes**  
- Add **spacing (margin, padding)**  
- Create **layouts and positioning**  
- Add **animations and effects**  

---

## 🔷 Why Use CSS?

- Makes websites **visually attractive**  
- Separates **content (HTML)** from **design (CSS)**  
- Improves **user experience**  
- Allows **reusable styles** across pages  

---

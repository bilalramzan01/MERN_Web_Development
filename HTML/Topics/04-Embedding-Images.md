# 📌 HTML `<img>` Tag

The `<img>` tag is used to **embed images** in a web page.

- It **does not have a closing tag** (self-closing).  
- It’s an **inline element**, meaning it flows along with text by default.

---

## 🔹 Syntax

```html
<img src="image.jpg" alt="Description of image" width="300" height="200">
<img src="images/photo.jpg" alt="Beautiful scenery">
<img src="https://example.com/image.jpg" alt="Example Image" width="400">
<img src="logo.png" alt="Company Logo" title="Our Logo">

# What is an HTML Tag?

An **HTML Tag** is a markup instruction written within angle brackets (`<` and `>`) that defines how a web browser should display content.

---

## 📌 Structure

Most HTML tags come in pairs:

- **Opening tag:** `<tagname>`
- **Closing tag:** `</tagname>`

---

## 📖 Examples of Paired Tags

- `<html> ... </html>`
- `<p> ... </p>` *(Paragraph)*
- `<h1> ... </h1>` *(Heading 1)*

---

## 🔹 Single Tags

Some tags are used as a single tag (without a closing tag). These are called **void** or **self-closing tags**.

---

## 📖 Examples of Single Tags

- `<br>` *(Line break)*
- `<hr>` *(Horizontal line)*

# HTML Heading Tags

HTML heading tags are used to define headings on a webpage.  
They range from `<h1>` to `<h6>`. base on their importance

---

## 📌 Structure

```html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

# HTML Comment Tag

The **HTML Comment Tag** is used to add notes or comments in the code.  
These comments are **not displayed in the web browser**.

---

## 📌 Syntax

```html
<!-- This is a comment -->

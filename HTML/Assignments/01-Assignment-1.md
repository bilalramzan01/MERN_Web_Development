# 📚 Assignment: HTML Practice

## 📝 Things To Do

### 1️⃣ View Website Source Code
- Open any website in your browser  
- Right-click and select **"View Page Source"**  

---

### 2️⃣ Copy and Save Code
- Copy the source code  
- Paste it into **Notepad**  
- Save the file with `.html` extension  
  - Example: `index.html`

---

### 3️⃣ Open in Browser
- Open the saved `.html` file in a browser  
- Observe the differences between:
  - Original website  
  - Your saved file  

---

## 🎯 Objective

- Understand how HTML works  
- Learn how browsers render web pages  
- Identify differences between live websites and static HTML files
